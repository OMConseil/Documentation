# Nomenclature

Éléments
```
e: Équipement
o: Organisation 
i: Infrastructure
d: Département
p: Profil
es: équipement de la société
```

Couples
```
(p,e): profil p utilisant un équipement e
(p,d): profil p appartenant à un departement d
(d,o): departement d appartenant à une organisation o
(e,i): equipement e appartenant à une infrastructure i
```
